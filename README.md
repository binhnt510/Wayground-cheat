
## How to Use

1. Open the quiz page.

2. Open **Developer Tools**:

   * Press `F12` or `Ctrl + Shift + I`
   * Go to the **Console** tab

3. Copy the code from the file **wayground.js**, paste it into the console, and press Enter

4. When prompted, enter the **PIN**

---

## Result

* The script will automatically fetch answers from the API.

* Correct answers will be highlighted:

  * 🟩 **Green border/background**
  * ⬛ **Black text**

* The highlighting updates automatically when new questions appear.

---

## Notes

* Do not modify the script before running.
* Make sure the PIN is correct.
* If nothing happens, the API may be unavailable or the quiz structure has changed.
